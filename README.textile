h2. Requirements

	* prototype "prototype.js":http://prototypejs.org
	* script.aculo.us "effects.js":http://script.aculo.us
	* graphika studios "droplicious.js":http://github.com/darrenterhune/droplicious
	* candy

h2. Install

	* you can download all the needed files right from here
	* <code>git clone git://github.com/darrenterhune/droplicious.git</code>
	* throw it all in a shaker, er your website dir
	* let it dry
